群体遗传学中的选择模式

Posted on 2020-02-18 | In Population Genetics

在群体遗传学中，假设SNP的两个等位基因为A和a，三种基因型AA，Aa和aa的适合度分别为ωAA,ωAa和ωaa，如果三种基因型的适合度不相等，则表明有选择的发生。

如果 ωAA > ωAa > ωaa 或者 ωAA < ωAa < ωaa，则表明存在定向选择(directional selection)，定向选择通过提高或者降低等位基因的频率而倾向于消除群体中的变异。
如果群体中杂合子的适合度最高，即 ωAA < ωAa > ωaa，则表明存在超显性(overdominance)。超显性属于平衡选择(balancing selection)的一种，后者指的是通过选择保持群体的变异性(variability)。
在分子进化中，选择的模式有正向选择(positive selection) 、负向选择(negative selection) 、净化选择(purifying selection)和歧化选择(diversifying selection) 。

  • 正向选择指的是新变异的出现是有利的而对有利变异进行选择，定向选择和超显性都属于正向选择的一种;
  • 负向选择指的是新变异的出现是有害的，有害变异被选低或者淘汰的过程，负向选择等同于净化选择;
  • 歧化选择指的是群体中两个或者多个极端的表型同时被选择的过程，歧化选择也增加群体的变异性。

自然状况下随机变异更倾向于产生不利变异，所以许多新变异的出现更容易遭受负向选择而从基因库中移除。当一个新变异出现但不影响个体的适合度时，即 ωAA = ωAa = ωaa，则该位点被认为是中性的，该位点不受选择影响， 统计学中拒绝中性进化模型的检验被称为中性检验 。
